# Extension Icons

Place your extension icons here:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create simple icons or use an icon generator. For now, the extension will work without icons, but Chrome will show a default icon.

To create icons quickly, you can:
1. Use an online icon generator
2. Create simple PNG files with a weather icon (cloud, sun, etc.)
3. Use a tool like Figma or Canva to design them
